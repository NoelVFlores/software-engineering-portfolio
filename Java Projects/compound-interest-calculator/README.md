# 💰 Compound Interest Calculator (Java)

This is a simple **Java-based command-line program** that calculates compound interest over time. The user inputs the principal amount, interest rate, number of times interest is compounded per year, and the duration in years. The program then outputs the final amount using the compound interest formula.

## 📈 Formula Used

The formula for compound interest is:
A = P(1 + r/n)^(nt)

Where:
- **A** = Final amount
- **P** = Principal amount (initial investment)
- **r** = Annual interest rate (decimal)
- **n** = Number of times interest is compounded per year
- **t** = Number of years


### 💻 Requirements

- Java Development Kit (JDK) installed
- Terminal or IDE (IntelliJ)

# Output Example - 
Enter the principal amount: 1000
Enter the interest rate (in %): 5
Enter the # of times compounded per year: 4
Enter the number of years: 10
The amount after 10 years is: $1,648.66
