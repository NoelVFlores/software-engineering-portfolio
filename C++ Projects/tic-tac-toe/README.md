# 🕹️ Tic Tac Toe Game in C++

This is a simple command-line Tic Tac Toe game built in C++ as a personal project. The game features a human player vs. computer mode with randomized computer moves. It runs in the terminal and allows the user to place markers (X or O) on a 3x3 board until a win or tie condition is met.

## 🎮 Gameplay Features

- Turn-based play between a **human player** and a **computer opponent**
- Randomized computer moves using the `rand()` function and `ctime` seed
- Basic win/tie condition detection
- Simple ASCII board rendering in the console
- Game loop with immediate feedback on win/loss/tie

## 🧠 Concepts Used

- Arrays and pointers
- Loops and conditionals
- Functions and modular code
- Random number generation
- Console input/output
- Basic AI logic (randomized opponent)

## 🛠️ How to Compile and Run

### 💻 Requirements

- A C++ compiler (like `g++`)
- A terminal or IDE (e.g., Visual Studio Code)
